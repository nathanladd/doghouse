Exercise 1: Verify Proper Operation

Practice reading and interpreting AC system pressures. Install a gauge set and lower the cab if necessary. Start the machine and operate af half throttle. AC on full. Give it 5 minutes.

  • Document temps and feel the amount of air flow at the ducts. Compare upper to lower ducts.
  • Document low and high side pressures
  • Compare readings to 8 times rule of thumb
  • Document cycle time. Stopwatch the on-time versus the off-time of the compressor
Equipment needed: Manifold gauges, Thermometer, Stopwatch
Exercise 2: Low Charge

Practice reading and interpreting AC system pressures. Using a recycling machine, recover, then recharge only 0.5 lbs of refrigerant into the machine

  • Connect gauge set and operate machine at half throttle with AC on full
  • Record high and low side pressures
  • Record compressor cycle times
  • Record duct temperature
Equipment needed: AC recycling machine, Thermometer, Stopwatch
Exercise 3: Airflow Testing

Practice reading and interpreting AC system pressures. Block the condenser with cardboard. Install a gauge set. Operate the machine at half throttle, AC on full.

  • Measure duct temperatures
  • Observe gauge readings and measure temperature drop across condenser.
  • Move blockage to cab recirc filter - observe pressure changes
Equipment needed: Thermometer, Bobcat machine, clean filters
Exercise 4: Electrical Diagnostics

Practice electrical troubleshooting on Bobcat AC systems.

  • Bypass compressor clutch using high-current test light
  • Bypass pressure switch function (cuts out below 28 psi / above 384 psi on high side)
  • Voltage drop power circuit to MCC
Equipment needed: Multimeter, test leads, High-current test light,wiring diagrams
Exercise 5: Heater Valve Leaking

A common failure. The AC is working correctly, however, the heater core is hot and the AC cannot overcome it. Turn the HVAC temperature control switch to medium. Install a gauge set. Start the machine and operate af half throttle. AC on full. Give it 5 minutes.

  • Measure the duct temperature and notice the air flow amount.
  • Install a gauge set. Note the compressor cycle times.
  • Feel the heater core inlet and outlet lines.
